## 📊 文件统计

- `public/api/user/` - 用户 API

## 🎯 快速导航

### 开发者入口
```bash
# 1. 数据库迁移
mysql -u root -p jef_analytics < database/migrations/001_add_user_system.sql

# 2. 迁移 admin 账户
php database/migrations/migrate_admin.php

# 3. 访问登录页面
http://localhost/login.php
```

### 测试流程
```
1. 注册 → /register.php
2. 登录 → /login.php
3. 管理站点 → /sites.php
4. 个人设置 → /profile.php
5. 退出 → /api/auth/logout.php
```

### 阅读文档顺序
```
1. 项目文件清单.md (本文档)
2. requirements.md (了解需求)
3. design.md (了解架构)
4. IMPLEMENTATION_COMPLETE.md (完成总结)
5. P0/P1/P2 修复报告 (详细优化)
```
