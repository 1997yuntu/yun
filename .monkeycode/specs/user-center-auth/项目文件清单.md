## 📝 使用方法

### 1. 初始化数据库
```bash
# 1. 执行基础表结构
mysql -u root -p jef_analytics < database/init.sql

# 2. 执行用户系统迁移
mysql -u root -p jef_analytics < database/migrations/001_add_user_system.sql

# 3. （可选）迁移旧版 admin 账户
php database/migrations/migrate_admin.php
```

### 2. 配置系统
```php
// 1. 确保 app/config.php 存在
// 2. 访问 /install/ 完成初始化
// 3. 在系统设置中开启/关闭用户注册
```

### 3. 测试功能
```

站点管理：http://localhost/sites.php
个人设置：http://localhost/profile.php
```
